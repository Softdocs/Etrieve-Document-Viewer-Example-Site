## Example Site Setup

### Install Node Modules

Run the following command to download node modules.  

``` powershell
yarn
```

Or if you use NPM

``` powershell
npm install
```

### Run

Run the following command to build and run the site (port 9000).

``` powershell
yarn run run-watch
```

or if you use NPM

``` powershell
npm run-script run-watch
```
